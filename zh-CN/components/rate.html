Rate 评分 ​

用于评分

基础用法 ​

评分默认被分为三个等级，可以利用颜色数组对分数及情感倾向进行分级（默认情况下不区分颜色）。 三个等级所对应的颜色用 colors 属性设置，而它们对应的两个阈值则通过 low-threshold 和 high-threshold 设定。

默认不区分颜色

区分颜色

<template>
  <div class="demo-block">
    <p>默认不区分颜色</p>
    <tu-rate v-model="value1" />
  </div>
  <div class="demo-block">
    <p>区分颜色</p>
    <tu-rate v-model="value2" :colors="colors" />
  </div>
</template>

<script lang="ts" setup>
import { ref } from 'vue';

const value1 = ref(null);
const value2 = ref(null);
const colors = ref(['#99A9BF', '#F7BA2A', '#FF9900']); // 等同于 { 2: '#99A9BF', 4: { value: '#F7BA2A', excluded: true }, 5: '#FF9900' }
</script>

<style scoped>
.demo-block {
  width: 50%;
  display: inline-block;
}
</style>

辅助文字 ​

用辅助文字直接地表达对应分数。为组件设置 show-text 属性会在右侧显示辅助文字。 通过设置 texts 可以为每一个分值指定对应的辅助文字。 texts 为一个数组，长度应等于最大值 max。

<template>
  <tu-rate
    v-model="value"
    :texts="['oops', 'disappointed', 'normal', 'good', 'great']"
    show-text
  />
</template>

更多种类的图标 ​

当有多层评价时，可以用不同类型的图标区分评分层级。设置icons属性可以自定义不同分段的图标。 若传入数组，共有 3 个元素，为 3 个分段所对应的类名；若传入对象，可自定义分段，键名为分段的界限值，键值为对应的类名。 本例还使用 void-icon 指定了未选中时的图标类名。

<template>
  <tu-rate
    v-model="value"
    :icons="icons"
    :void-icon="ChatRound"
    :colors="['#409eff', '#67c23a', '#FF9900']"
  />
</template>

<script lang="ts" setup>
import { ref } from 'vue';
import {
  ChatDotRound,
  ChatLineRound,
  ChatRound
} from '@tu-view-plus/icons-vue';

const value = ref();
const icons = [ChatRound, ChatLineRound, ChatDotRound]; // 等同于 { 2: ChatRound, 4: { value: ChatLineRound, excluded: true }, 5: ChatDotRound }
</script>
<template>
  <tu-rate
    v-model="value"
    :icons="icons"
    :void-icon="ChatRound"
    :colors="['#409eff', '#67c23a', '#FF9900']"
  />
</template>

<script lang="ts" setup>
import { ref } from 'vue';
import {
  ChatDotRound,
  ChatLineRound,
  ChatRound
} from '@tu-view-plus/icons-vue';

const value = ref();
const icons = [ChatRound, ChatLineRound, ChatDotRound]; // 等同于 { 2: ChatRound, 4: { value: ChatLineRound, excluded: true }, 5: ChatDotRound }
</script>

只读 ​

只读的评分用来展示分数， 允许出现半星。为组件设置 disabled 属性表示组件为只读。 此时若设置 show-score，则会在右侧显示目前的分值。 此外，您可以使用属性 score-template 来提供评分模板。 模板为一个包含了 {value} 的字符串，{value} 会被替换为当前分值。

3.7 points
<template>
  <tu-rate
    v-model="value"
    disabled
    show-score
    text-color="#ff9900"
    score-template="{value} points"
  />
</template>

<script lang="ts" setup>
import { ref } from 'vue';

const value = ref(3.7);
</script>
<template>
  <tu-rate
    v-model="value"
    disabled
    show-score
    text-color="#ff9900"
    score-template="{value} points"
  />
</template>

<script lang="ts" setup>
import { ref } from 'vue';

const value = ref(3.7);
</script>

自定义样式 ​

可以为rate组件设置自定义样式。 使用 css/scss 语言来更改全局或局部颜色。 设置了一些全局颜色变量：--rate-void-color、--rate-fill-color、--rate-disabled-void-color 和 --rate-text-color。 可以类似这样使用 :root { --rate-void-color: red; --rate-fill-color: blue; } 。
