BackTop API ​

BackTop Attributes ​

参数名描述类型默认值
visible-height显示回到顶部按钮的触发滚动高度Number200
target触发滚动的对象string-
right控制其显示位置，距离页面右边距Number30
bottom控制其显示位置，距离页面底部距离Number20

BackTop Events ​

事件名描述参数
click点击按钮触发的事件Function
