Alert ​

Displays important alert messages.

Basic usage ​

Display information that needs attention, suitable for brief warning prompts.

This is an default alert.
<template>
  <tu-alert title="This is an default alert." />
</template>
<template>
  <tu-alert title="This is an default alert." />
</template>

Types ​

There are four types of warnings: info, success, warning, and error. which has no icon by default.

This is an success alert.
This is an warning alert.
This is an error alert.
This is an info alert.
<template>
  <tu-alert title="This is an success alert." type="success" effect="light" />
  <tu-alert title="This is an warning alert." type="warning" effect="light" />
  <tu-alert title="This is an error alert." type="error" effect="light" />
  <tu-alert title="This is an info alert." type="info" effect="light" />
</template>

<style>
.tu-alert:not(:last-child) {
  margin-bottom: 20px;
