Spin 加载中 ​

何时使用 ​

用于页面和区块的加载中状态 - 页面局部处于等待异步数据或正在渲染过程时，合适的加载动效会有效缓解用户的焦虑。

基本用法 ​

用于展示加载中的状态。

<template>
  <tu-spin :loading="true" />
</template>
<template>
  <tu-spin :loading="true" />
</template>

不同尺寸 ​

设置 size 可以得到不同尺寸的加载图标。

<template>
  <tu-spin :loading="true" :size="10" />
  <tu-spin :loading="true" class="ml-4" />
  <tu-spin :loading="true" :size="18" class="ml-4" />
</template>
<template>
  <tu-spin :loading="true" :size="10" />
  <tu-spin :loading="true" class="ml-4" />
  <tu-spin :loading="true" :size="18" class="ml-4" />
</template>

点类型指示符 ​

通过 dot 属性，可以展示点类型的指示符。

<template>
  <tu-spin :loading="true" :size="16" dot />
</template>
<template>
  <tu-spin :loading="true" :size="16" dot />
</template>

可以给任意内容添加加载中指示符。

添加描述文案 ​

通过 tip 属性添加描述文案。

加载中..
<template>
  <tu-spin :loading="true" tip="加载中.." />
</template>
<template>
  <tu-spin :loading="true" tip="加载中.." />
</template>

自定义图标 ​

通过 #icon 插槽可以自定义图标。

<template>
  <tu-spin :loading="true" :size="20">
    <template #icon>
      <Loading />
    </template>
  </tu-spin>
</template>

<script lang="ts" setup>
import { Loading } from '@tu-view-plus/icons-vue';
</script>
<template>
  <tu-spin :loading="true" :size="20">
    <template #icon>
      <Loading />
    </template>
  </tu-spin>
</template>

<script lang="ts" setup>
import { Loading } from '@tu-view-plus/icons-vue';
</script>
